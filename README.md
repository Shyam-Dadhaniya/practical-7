## User List App
#### 1. Install NPM:

```sh
$ npm install
```

#### 2. Start NPM:

```sh
$ npm start
```

#### 3. Start Development:

```sh
$ npm run start:dev
```

#### 4. For Production Build:

```sh
$ npm run build:production
```

#### 5. For Staging Build:

```sh
$ npm run build:
```
#### 6. For Serve Build:

```sh
$ serve -s build
```


##### Open http://localhost: .... to view it in your browser.
